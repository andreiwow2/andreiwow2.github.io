How?

1.

Wenn Sie gefragt werden, erlauben Sie Ihrem Browser, externe Links zu öffnen.

Nur die Registerkarte "Content Management" ist zu verwenden.
Sobald Sie sich auf der Registerkarte "Content Management" befinden, richtet sich Ihr Interesse auf die folgenden Registerkarten: "InfoArticle", "Notification", "StoreLocation", "FeaturedArticle".

InfoArticle: Hier können Sie Nachrichtenartikel erstellen, die dann in der App angezeigt werden.

FeaturedArticle: Wenn Sie einen InfoArticle erstellt haben, können Sie ihn hier auswählen, damit er als Hauptartikel in der App angezeigt wird.

Notification: Hier können Sie Benachrichtigungen erstellen, die an Geräte gesendet werden, die Benachrichtigungen akzeptieren. Nach der Erstellung dauert es ein wenig, bis die Benachrichtigung zugestellt wird.

StoreLocation: Hier können Sie die Standorte unserer Geschäfte hinzufügen, die in der App angezeigt werden sollen.
